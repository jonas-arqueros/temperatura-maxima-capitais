# Temperatura Máxima das Capitais Brasileiras

## Objetivo

Analisar a evolução da temperatura máxima diária em capitais brasileiras utilizando dados históricos do INMET.

## Perguntas

- Quais capitais apresentam as maiores temperaturas máximas médias?
- Houve aquecimento ao longo das últimas décadas?
- A frequência de dias extremamente quentes aumentou?
- Existem diferenças regionais?

## Ferramentas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Fonte dos Dados

Instituto Nacional de Meteorologia (INMET)
